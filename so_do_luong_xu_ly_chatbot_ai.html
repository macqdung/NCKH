Sơ đồ hoạt động xử lý chatbot AI

Đây là sơ đồ hoạt động theo đúng luồng nghiệp vụ bạn mô tả: người dùng gửi tin nhắn, PHP Backend trung chuyển bằng AJAX/API nội bộ, Python Chatbot Service xử lý ngữ cảnh, gọi Gemini API, truy vấn MySQL khi cần, lưu lịch sử chat và trả phản hồi ngược về website.

BẮT ĐẦU
1

Người dùng gửi tin nhắn

Người dùng nhập câu hỏi vào giao diện chatbot trên website và nhấn gửi. Trình duyệt chuẩn bị dữ liệu gồm nội dung tin nhắn và thông tin phiên nếu có.

2

Gửi đến PHP Backend

Frontend gửi tin nhắn đến PHP Backend thông qua AJAX. PHP đóng vai trò lớp trung gian nhận request từ website trước khi chuyển tiếp sang dịch vụ AI nội bộ.

3

PHP Backend gọi Python Chatbot Service

PHP Backend gửi tin nhắn của người dùng cùng ID phiên chat tới Python Chatbot Service qua API nội bộ, ví dụ một REST API dùng JSON.

4

Python Chatbot Service xử lý

Dịch vụ Python nhận request và thực hiện toàn bộ pipeline xử lý hội thoại trước khi tạo câu trả lời cuối cùng.

Duy trì ngữ cảnh bằng cách đọc lịch sử trò chuyện từ bảng chat_messages theo ID phiên. Gửi tin nhắn người dùng cùng ngữ cảnh hội thoại tới Gemini API. Nếu nội dung liên quan sản phẩm, thực hiện truy vấn các bảng books và categories. Tổng hợp phản hồi cuối cùng, sau đó lưu cả tin nhắn người dùng và câu trả lời bot vào chat_messages.
?

Câu hỏi có liên quan dữ liệu sản phẩm?

Ví dụ: sách khoa học viễn tưởng, giá sách, danh mục sách, gợi ý theo chủ đề.

4A

Gọi Gemini API

Python gửi nội dung người dùng và ngữ cảnh hội thoại đến Google Gemini API để xử lý ngôn ngữ tự nhiên và sinh câu trả lời.

4B

Truy vấn MySQL nếu cần

Nếu câu hỏi liên quan đến sản phẩm, Python truy vấn bảng books, categories và đồng thời dùng chat_messages để đọc hoặc lưu lịch sử trò chuyện.

5

Python tổng hợp và trả phản hồi

Dịch vụ Python kết hợp nội dung do Gemini sinh ra với dữ liệu từ CSDL nếu có, lưu lịch sử chat vào chat_messages, rồi gửi phản hồi hoàn chỉnh về PHP Backend.

6

PHP Backend chuyển tiếp kết quả

PHP nhận phản hồi từ Python Chatbot Service và trả lại dữ liệu cho frontend theo định dạng mà website đang sử dụng.

7

Hiển thị cho người dùng

Giao diện website nhận phản hồi từ PHP Backend và hiển thị câu trả lời của chatbot cho người dùng trong khung hội thoại.

KẾT THÚC

Tóm tắt luồng

Người dùng nhập tin nhắn trên website, AJAX gửi sang PHP Backend, PHP gọi Python Chatbot Service. Python lấy ngữ cảnh từ chat_messages, gọi Gemini API, truy vấn books và categories khi cần, tạo phản hồi, lưu lịch sử chat, rồi trả kết quả ngược lại qua PHP để frontend hiển thị cho người dùng.
